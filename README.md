# My Banking Web Application

A secure banking portal with login authentication and dashboard features.

## Features
- 🔐 User Login/Logout
- 📊 Account Dashboard  
- 💰 Loan Management
- 📝 Foreclosure Application
- 🎨 Responsive Design

## Technologies Used
- HTML5
- CSS3
- JavaScript

## How to Run
1. Download index.html
2. Open in any web browser
3. Use demo accounts to login
